Definition List of Computer Parts

CPU (Central Processing Unit)
The brain of the computer where most calculations take place.
RAM (Random Access Memory)
Temporary memory that stores data currently in use.
Motherboard
The main circuit board connecting all components of a computer.
Power Supply Unit (PSU)
Converts electrical energy to the appropriate voltage for the computer.
Hard Drive (HDD) / Solid State Drive (SSD)
Permanent storage for files and applications.
Graphics Processing Unit (GPU)
Handles rendering of images, videos, and animations.
Cooling System
Prevents the computer from overheating, including fans and liquid cooling.
Optical Drive
Reads and writes CDs, DVDs, and Blu-ray discs.
Network Interface Card (NIC)
Enables the computer to connect to the internet or local networks.
BIOS/UEFI Firmware
Basic software that initializes hardware during startup.
